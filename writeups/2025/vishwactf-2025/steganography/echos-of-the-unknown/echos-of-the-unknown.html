# Echos of the Unknown Challenge description: > Not everything is heardâ€”some things must be seen. Alrighty, so for this challenge we are given a `wav` file that appears to have some sort of morse code signal inside of it. Now, while it is possible to manually decode that by hand, its the 21st century for god's sake. So lets use one of my favorite tools for this, [MorseCode.world](https://morsecode.world/international/decoder/audio-decoder-adaptive.html)! However, it seems that the file we have is too slow, so lets bump up the speed of it. You can use whatever audio/video editor floats your boat, but once you speed it up the online transcriber will work way more effectively. ![The decoding in process](assets/image0.png) FLAG: `VishwaCTF{CR4CK3D_7H3_C0D3}`
